Мои комнаты

Ученик

Ученик входит по 4-значному коду и имени.

Формат вопросов

Для MVP загружается `JSON`-массив. Каждый вопрос двигает ученика по полю.

[
  {
    "prompt": "Столица Франции",
    "answer": "Париж",
    "points": 3
  },
  {
    "prompt": "2 + 2",
    "answer": "4",
    "points": 1
  }
]
